大象 App 介绍

大象 App 是一款 PHPHub 社区第三方 Android 客户端, 该 App 界面设计遵循了 Google 的 Material Design 设计风格, 使用了 MVP 架构模式, 使用 RxJava + Retrofit 对网络数据行处理.

欢迎大家使用, 如果发现 bug 可通过 App 意见反馈, 或者通过微博以及 GitHub与我取得联系.

使用到的开源库

  • Logger
    一个漂亮的Log输出日志，支持json格式化输出
  • Material-Dialogs
    一个漂亮的Material Dialog
  • BGABadgeView-Android
    类似微博徽章
  • MultiStateView
    通用显示各种状态 View
  • glide-transformations
    快速实现毛玻璃效果
  • FloatingActionButton
    格式化日期时间
  • barcodescanner
    二维码扫描
  • fresco
    Facebook 开源的图片缓存库
  • butterknife
    注解框架
  • materialish-progress
    Material Design 风格进度条
  • PhotoDraweeView
    基于 Fresco 的图片缩放控件
  • cwac-anddown
    Markdown 渲染
  • gm-mkdroid
    一个所见即所得的 Markdown 编辑器
  • android-Ultra-Pull-To-Refresh-With-Load-More
    这个是基于 Ultra-Pull-to-Refresh 库修改增加支持加载更多的刷新控件
  • T-MVP
    泛型深度解耦下的MVP大瘦身
